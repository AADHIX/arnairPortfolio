Design and Fabrication of Eye Blink Based Automatic Pneumatic Bumper and Braking System

Eye blink detection to activate an automatic braking and bumper deployment mechanism in emergency situations

Project Overview

This project aims to enhance automotive safety by developing an intelligent system that uses eye blink detection to activate an automatic braking and bumper deployment mechanism in emergency situations. The system is particularly beneficial in preventing accidents caused by driver drowsiness or sudden loss of attention.

Key Features

  • Real-time Drowsiness Detection - Uses IR-based eye blink sensor to monitor driver alertness.
  • Automatic Braking System - Engages brakes when prolonged eye closure (>2 sec) is detected.
  • Pneumatic Bumper Deployment - Extends a shock-absorbing bumper to minimize collision impact.
  • Multi-Stage Safety Alert - Warns driver via buzzer/vibration before activating brakes & bumper
  • False Trigger Prevention - Adjustable delay and sensitivity settings to avoid unnecessary activation.
  • Low-Cost & Scalable - Uses Arduino-based control for easy integration in most vehicles.

Technical Specifications

  • Controller: Arduino Uno (ATmega328P, 16MHz, 32KB Flash)
